# 求职展示要点

## 简历项目描述

手势交互游戏合集：基于 OpenCV、MediaPipe 和 Pygame 实现摄像头实时手势识别，将手部关键点映射为游戏操作，完成手势启动器、贪吃蛇和俄罗斯方块两个可交互游戏。负责手势检测封装、游戏状态管理、碰撞检测、中文 UI 绘制、音频资源管理和 GitHub 工程化整理。

## 可讲技术点

- 使用 MediaPipe Hands 获取 21 个手部关键点，封装左右手识别、五指伸展状态和指尖距离计算。
- 将连续的视频帧输入转化为稳定的离散游戏指令，降低误触并支持悬停蓄力按钮。
- 在贪吃蛇中实现食指轨迹控制、平滑蛇头、食物/障碍物碰撞、自身碰撞和最高分记录。
- 在俄罗斯方块中实现网格状态、方块旋转、边界检测、消行计分、影子方块和双手协作控制。
- 兼顾 OpenCV 和 Pygame 两类渲染方式，处理中文字体、窗口生命周期、摄像头释放和音频清理。

## 面试回答提示

**为什么做这个项目？**

想把计算机视觉和传统小游戏结合起来，做一个不用键盘鼠标也能玩的交互 demo。相比单纯调用模型，这个项目更强调实时性、交互状态管理和用户体验。

**最大的难点是什么？**

手势识别结果是连续且有噪声的，不能直接等同于游戏操作。项目里通过手指组合、悬停计数、冷却时间和平滑坐标等方式，把不稳定的视频输入转成相对稳定的控制指令。

**如果继续优化，会怎么做？**

优先做手势校准、截图/GIF 展示、模块拆分和自动化测试；之后可以打包为 Windows 可执行文件，或者加入更多小游戏作为统一手势交互平台。
